# Anykey for Commodore Computers (and MEGA65)

There is a separate page for [Commodore PET](Commodore%20PET.md).

## Supported Models

### Anykey C64
![](Anykey%20C64.png)
- Commodore 64
- Commodore 128 (in C64 mode)
- MEGA65 (in C64 mode)

### Anykey C128
![](Anykey%20C128.png)
- Commodore 128 (in C128 40 columns mode)

### Anykey MEGA65
![](Anykey%20MEGA65.png)
- MEGA65 (in native mode)
  
### Anykey Plus/4
![](Anykey%20Plus-4.png)
- Commodore Plus/4
